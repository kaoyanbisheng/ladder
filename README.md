# 高中综评数据自动化核查工具 (Student Evaluation Checker)

这是一个为高中综评管理人员设计的自动化工具，用于快速筛查综评系统导出表格中未完成填报的学生名单。

## 📋 功能特点
1. **简单易用**：无需编程知识，网页界面操作。
2. **灵活配置**：自动识别表头，可手动勾选需要核查的“必填项”。
3. **精准定位**：自动生成未完成名单，列出具体缺失的项目。
4. **一键导出**：支持导出 Excel 核查报告，方便下发。

## 🛠️ 安装与运行

### 前置要求
您需要在电脑上安装 Python。

### 1. 安装依赖
在项目文件夹下打开命令行（CMD 或 PowerShell），运行以下命令安装必要的库：
```bash
pip install -r requirements.txt
```

### 2. 启动工具
运行以下命令启动程序：
```bash
streamlit run app.py
```
启动后，浏览器会自动打开工具界面（通常是 `http://localhost:8501`）。

## 📖 使用指南
1. **上传文件**：点击界面上的上传按钮，选择综评系统导出的 Excel 文件。
2. **配置列**：
    - **身份列**：选择用来识别学生的列（如：姓名、班级、学号）。
    - **必填列**：勾选必须填写的列（如：思想品德、社会实践）。
3. **开始核查**：点击“开始核查”按钮。
4. **下载结果**：如果有学生未完成，界面会显示列表，点击“下载未完成名单核查表”即可保存结果。

## ❓ 常见问题
- **文件读取失败？** 请确保上传的是标准的 Excel 文件 (.xlsx 或 .xls)，且没有加密。
- **如何测试？** 如果没有数据，可以勾选界面底部的“生成测试数据”，下载一个模板文件进行测试。
